# HyperOS Enchanter 🌟

![HyperOS Enchanter](https://via.placeholder.com/800x400.png?text=HyperOS+Enchanter)

**HyperOS Enchanter** — это продвинутый менеджер обоев и тем для устройств Xiaomi на базе HyperOS (и MIUI) с искусственным интеллектом под капотом. 
Благодаря удобному интерфейсу вы можете мгновенно создавать потрясающие обои с идеальными масками глубины (чтобы часы на экране блокировки прятались за объектом) и управлять темами прямо на вашем Root-устройстве!

---

## 🎨 Особенности

- **Генерация маски нейросетью:** Использует передовые ИИ модели (`isnet-general-use`, `u2net`, `birefnet-general`) для идеального вырезания объекта и создания маски глубины.
- **Интеграция с Root ADB:** Автоматически закидывает сгенерированные обои и правит файл `wallpaper_config.json` прямо в системной папке `/product/media/wallpaper` без необходимости писать команды вручную.
- **Менеджер тем:** Просматривайте, переименовывайте и удаляйте темы и обои прямо с ПК!
- **Свои маски:** Нужна абсолютная точность? Вы можете загрузить заранее подготовленную маску в виде картинки!
- **Поддержка языков:** Полный перевод на русский и английский (переключается в настройках).
- **Сборка в EXE:** Есть возможность автоматически собрать портативное `.exe` приложение для Windows, чтобы не зависеть от Python!

## 🚀 Установка и Требования

### Для обычных пользователей (Windows)
1. Скачайте исходный код.
2. Запустите файл `scripts\install_exe.bat`. 
3. Скрипт сам создаст виртуальное окружение, скачает всё необходимое и соберет полноценный `.exe` файл.
4. Зайдите в папку `dist/HyperOS_Enchanter` в корне проекта и запустите `HyperOS_Enchanter.exe`.

### Для разработчиков (Python)
Убедитесь, что у вас установлен Python 3.8+.

**Windows:**
```bat
cd scripts
install.bat
start.bat
```

**Linux/macOS:**
```bash
cd scripts
chmod +x *.sh
./install.sh
./start.sh
```

### Требования к телефону
- Ваше устройство Xiaomi/POCO должно быть **с ROOT-правами** (Magisk/KernelSU).
- Обязательно включите **Отладку по USB** в настройках разработчика.
- Подключите телефон к ПК кабелем и разрешите отладку на экране смарта.

## 🛠️ Как пользоваться

### Вкладка 1: Создание обоев
1. Нажмите **Выбрать фото** и загрузите картинку.
2. Подождите, пока ИИ высчитает маску (вы увидите красную заливку на оригинальном фото).
3. Если ИИ плохо "отрезал" горизонт или фон, выберите другую модель в выпадающем списке и нажмите **ИИ Маска**.
4. Введите желаемое имя для обоев в текстовое поле.
5. Нажмите **Установить на телефон (Root)**. При появлении запроса на экране телефона - предоставьте права суперпользователя (Root).

### Вкладка 2: Менеджер тем
1. Нажмите **Поиск всех тем на телефоне**.
2. Выберите папку с темой слева, чтобы увидеть список обоев.
3. Выберите обои по центру, чтобы посмотреть их превью (Оригинал + Маска).
4. Вы можете свободно переименовывать или удалять фото и целые темы в 1 клик!
